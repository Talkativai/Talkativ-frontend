# Talkativ UI 🗣️

Talkativ is an advanced AI Voice Agent management dashboard designed specifically for modern restaurants and businesses. The platform allows business owners to seamlessly configure, deploy, and monitor realistic AI voice assistants capable of handling reservations, processing orders, and answering customer FAQs autonomously.

## Core Features 🚀
- **Interactive KPI Dashboard:** View up-to-the-minute metrics on total calls, AI-handled orders, and booking statuses with quick navigational shortcuts to deep-link views.
- **Agent Identity Management:** Completely customize your AI Agent's persona, voice tone, spoken languages, and call routing behaviors in a sleek professional interface.
- **Knowledge Base Synchronization:** Seamlessly sync your business FAQs, operating hours, and live menu items directly to the AI's core brain allowing it to answer dynamic customer inquiries accurately.
- **Order & Booking Tracking:** Monitor accepted, out-for-delivery, and cancelled orders in real-time alongside waitlist and reservation management functionalities.
- **Streamlined Setup Flow:** A premium 5-step onboarding wizard designed to get restaurants live with an AI agent in minutes.

## Tech Stack

- Vite
- Modern Vanilla CSS (Custom Design System)
- Phosphor Icons / Emojis
